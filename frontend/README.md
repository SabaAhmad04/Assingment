# Dynamic Form Builder (MERN + Airtable Integration)

A full-stack dynamic form builder using **React (Vite), Express, MongoDB** and **Airtable OAuth**.  
Users can log in, create custom forms using Airtable fields, add conditional logic, and store responses in **MongoDB + Airtable**.



 Frontend   :- https://assingment-fuz1m3ekv-saba-ahmads-projects-070c3af5.vercel.app
 Backend API :- https://assingment-4.onrender.com/ 


Frontend: React (Vite), React Router  
Backend: Node.js, Express, MongoDB (Mongoose)  
Auth: Airtable OAuth  
Deployment: Vercel (frontend), Render/Railway (backend)
